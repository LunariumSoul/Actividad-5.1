1.-Los 3 archivos contienen los ejemplos modificados de la pagina de dart, utilizando listas, maps, sets, ciclos, condiciones y menus

2.-Para realizarlos usé la extension de dart en visual studio code, para instalar el SDK use la consola de powershell de windows y usé el comando "choco install dart-sdk".

3.-Y para ejecutar los códigos se usa la misma terminal de VisualSudioCode, por ejemplo: " dart EjerciciosList.dart"

Nota: Asegúrate que desde la consola este en tu directorio donde están tus archivos a ejecutar.
